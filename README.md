# 과제 2 Mine Sweeper

- 패키지 설치 및 실행 방법은 아래와 같습니다. (현재 위치 assignments/mine-sweeper 기준)

  1. 패키지를 설치합니다.

  ```cmd
    npm install
  ```

  2. 프로젝트를 개발 환경에서 실행합니다.

  ```cmd
    npm run dev
  ```

  3. 로컬호스트에서 확인합니다.

  - Vite 기본 포트가 적용되는 경우, http://localhost:5173/ 에서 확인할 수 있습니다.

  - 해당 포트를 사용 중인 경우, 터미널에 표시된 포트로 접속합니다.
